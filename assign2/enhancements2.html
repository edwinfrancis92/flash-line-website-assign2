Enhancement 1 - Dynamically Changing Meal Selection Form

The Select Meal Combo Selection drop down menu has been designed to dynamically change two other drop down menus:

  • Select Sides Drop Down Menus
  • Select Drink Drop Down Menu

If you select a meal combo, it will unhide the two other drop down menus and clear any pre-exsisting options and proceed to fill the options up again with an array containing the right side dish and drink information assosiated with the combo. For example, if you click Big Belly lash Burger, it will change the drop down menu options below to have Chili Cheese Flash Fries and A bottle of Harry's beer as an option.
This Enhancement also changes the value of the drop down option. Ensuring accuracy of the form's name-value pairs.

This was made with a combination of using nested loops and selection to transfer the values of two dimensional arrays to the options element and using a for loop to go through old data and clear them.
While some of the clearing code was inspired from a answer to a question on Stackoverflow and Somacon.

Source:
  • How do I clear all options in a dropdown box?- Stackoverflow
  • Javascript Clear Select Options Fast - Somacon

Enhancement 2 - Countdown timer on payment page

On the payment page I have created a count down timer that floats to the right and sticks. It gets the current date and adds 2 minutes (in milliseconds) to the current date and compares that to another instance of the current date as a target to count down to.

There, a set of calculations that will convert the date data to a readable format of Minutes and Seconds to be read again.

This method was heavily inspired from various Stackoverflow forums and a blogpost. A big part of code came from one particular stackoverflow forum (the first listed below) but the code was modified to suit the website's needs.

Source:
  • JavaScript countdown timer for hour, minutes and seconds when a start button click - Stackoverflow
  • Build a Countdown Timer in Just 18 Lines of JavaScript - Sitepoint
  • Converting milliseconds to minutes and seconds with JavaScript - Stackoverflow
